# 라즈베리파이를 활용한 미니 프로젝트
추가 설명
## 제목: 출입문 경보 시스템
## 작품 개요:
LED, 온도 센서, 부저를 사용하는 프로젝트
안전하지 않은 온도를 감지할 경우 경고등과 알림음을 재생
주택, 사무실 등 건물내에서 출입문의 안전을 강화하는데 사용가능

## 작품 제작에 사용한 센서와 엑추에이터와 역할
온습도 센서 : 주변 기온값을 입력받아 위험상황을 감지한다
LED : 온도가 올라가면 점등하여 경고를 알린다
부저 : 온도가 올라가면 알림음을 출력하여 경고를 알린다
